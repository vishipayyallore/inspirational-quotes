# Grammar Foundations

This folder provides structured lessons on essential English grammar concepts, each in its own file.

## Concepts Covered

- Nouns
- Pronouns
- Adjectives
- Articles
- Verbs
- Infinitives
- Gerunds
- Participles
- Tenses
- Adverbs
- Prepositions
- Conjunctions
- Interjections
- Sentences
- Voice
- Speech
- Comprehensions
- Compositions

Each lesson includes:

1. Definition and explanation of the concept.
2. Example sentences.
3. Practice exercises.
