# Cline Skills — Inspirational Quotes & English Learning

This directory contains repeatable procedures and automated checks for maintaining quality in the inspirational-quotes repository.

## Available Skills

### Content Verification
- `ci-checks.md` — Automated and manual checks for content quality, vocabulary ordering, and format compliance

### Documentation Verification
- `docs-verification.md` — Checks for documentation completeness and adherence to standards

### Workspace Management
- `workspace-review.md` — Comprehensive workspace health checks and maintenance procedures

### Topic Management
- `topic-companions.md` — Procedures for managing related content across different sections

## Usage

Skills are designed to be run by Cline agents when performing content creation, verification, or maintenance tasks. Each skill file contains:

- **Purpose**: What the skill accomplishes
- **Prerequisites**: Required tools and setup
- **Execution**: Step-by-step commands or procedures
- **Failure handling**: Guidance for addressing issues

## Quality Standards

All skills align with the repository's quality assurance standards defined in `.clinerules/rules/04-quality-assurance.md`.

## Customization

Skills can be customized to meet specific project needs while maintaining alignment with core educational principles and content standards.
