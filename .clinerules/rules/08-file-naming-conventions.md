# File Naming Conventions

## Quote Files
- Location: `InspirationalQuotes/`
- Format: Theme-based naming with .md extension
- Examples: `motivation.md`, `leadership.md`, `perseverance.md`
- Use lowercase with hyphens for multi-word filenames

## Vocabulary Files
- Location: `EnglishLearning/Vocabulary/`
- Format: Single uppercase letter followed by .md
- Examples: `A.md`, `B.md`, `C.md`
- Organized alphabetically

## Grammar Files
- Location: `EnglishLearning/Grammar/`
- Format: Descriptive names with .md extension
- Examples: `nouns-and-pronouns.md`, `verb-tenses.md`, `sentence-structure.md`
- Use lowercase with hyphens for multi-word filenames

## Foundation Files
- Location: `EnglishLearning/Foundations/`
- Format: Descriptive names with .md extension
- Examples: `basic-sentence-structure.md`, `parts-of-speech.md`
- Use lowercase with hyphens for multi-word filenames

## Daily Learning Plan Files
- Location: `EnglishLearning/`
- Format: Date-based or descriptive naming with .md extension
- Examples: `daily-learning-template.md`, `sample-daily-plan.md`
- Use lowercase with hyphens for multi-word filenames

## Strategic Literacy Files
- Location: `Strategic-Literacy/`
- Format: Numbered with descriptive names and .md extension
- Examples: `01-today-is-mine.md`, `02-execution-discipline.md`
- Use lowercase with hyphens for multi-word filenames

## Script Files
- Location: `Scripts/`
- PowerShell scripts use .ps1 extension
- Examples: `Verify-Vocabulary.ps1`, `Check-Quotes.ps1`
- Use PascalCase for PowerShell script names

## Image Files
- Location: `docs/images/`
- Format: Descriptive names with .png or .jpg extension
- Examples: `growth.png`, `learning-progress.png`
- Use lowercase with hyphens for multi-word filenames

## Documentation Files
- Location: `docs/`
- Format: Descriptive names with .md extension
- Examples: `contributing.md`, `style-guide.md`
- Use lowercase with hyphens for multi-word filenames
