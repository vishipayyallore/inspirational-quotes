# Agent instructions (Cline)

This repository is a personal workspace for collecting inspirational quotes, building English vocabulary, and structured English learning.

## Scope

Applies to: quote curation, vocabulary definitions, grammar learning, and English learning automation inside this repo. Does **not** apply to external repos, one-off shell sessions, or anything outside this inspirational-quotes project.

## Read first

1. `README.md` — repo purpose and current structure.
2. `.clinerules/rules/01-project-scope.md` — primary scope and guidelines.
3. `.clinerules/rules/02-content-standards.md` — content quality standards.
4. `.clinerules/rules/03-repository-structure.md` — file organization standards.

### Precedence on conflict

When the files above disagree, resolve in this order (highest wins):

1. `.clinerules/rules/01-project-scope.md` (scope guardrail)
2. `.clinerules/rules/02-content-standards.md` (content quality)
3. `.clinerules/rules/03-repository-structure.md` (structure)
4. `README.md`
