# Cline Agents — Inspirational Quotes & English Learning

This directory contains delegated task playbooks and subagents for automated content validation and quality assurance.

## Available Agents

### Content Validation Agents
- `vocabulary-order-verify.md` — Automated verification of vocabulary alphabetical ordering
- `content-format-check.md` — Validation of quote and vocabulary entry formatting
- `quality-assurance-review.md` — Comprehensive content quality review agent

### Repository Management Agents
- `file-organization-audit.md` — Audit of file structure and naming conventions
- `markdown-standards-enforcement.md` — Enforcement of markdown formatting standards

## Usage

Agents are designed to be invoked by Cline for automated content validation, quality checks, and maintenance tasks. Each agent file contains:

- **Role**: Specific validation or auditing responsibility
- **Invocation criteria**: When the agent should be called
- **Execution procedure**: Automated steps for validation
- **Reporting format**: Standardized output for issues found

## Quality Standards

All agents align with the repository's quality assurance standards defined in `.clinerules/rules/04-quality-assurance.md`.

## Customization

Agents can be customized to meet specific project validation needs while maintaining alignment with core educational principles and content standards.
