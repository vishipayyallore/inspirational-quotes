# Cline Workflows — Inspirational Quotes & English Learning

This directory contains step-by-step workflows for common tasks and maintenance procedures in the inspirational-quotes repository.

## Available Workflows

### Content Creation Workflows
- `add-new-vocabulary.md` — Step-by-step process for adding new vocabulary words
- `add-new-quote.md` — Process for adding inspirational quotes with proper formatting
- `create-grammar-lesson.md` — Workflow for creating new grammar learning content

### Quality Assurance Workflows
- `run-ci-checks.md` — Complete content verification and quality checks
- `perform-quality-review.md` — Comprehensive quality assurance review process
- `verify-file-organization.md` — Audit of file structure and naming conventions

### Maintenance Workflows
- `update-vocabulary-order.md` — Process for reorganizing vocabulary files alphabetically
- `review-content-standards.md` — Periodic review of content quality and standards compliance
- `workspace-health-check.md` — Complete workspace maintenance and cleanup procedures

## Usage

Workflows are designed to guide Cline agents and human contributors through complex tasks with consistent, repeatable steps. Each workflow file contains:

- **Purpose**: What the workflow accomplishes
- **Prerequisites**: Required tools, permissions, or setup
- **Steps**: Detailed sequential instructions
- **Validation**: How to verify successful completion
- **Troubleshooting**: Common issues and solutions

## Quality Standards

All workflows align with the repository's quality assurance standards defined in `.clinerules/rules/04-quality-assurance.md`.

## Customization

Workflows can be customized to meet specific project needs while maintaining alignment with core educational principles and content standards.
