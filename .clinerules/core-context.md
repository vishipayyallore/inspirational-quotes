# Inspirational Quotes & English Learning — core context (Cline)

## Repository

**inspirational-quotes** — A personal workspace for collecting inspirational quotes, building English vocabulary, and structured English learning.

## Purpose

This repository serves to:
- Curate meaningful quotes for daily motivation and wisdom
- Build practical English vocabulary through contextual learning
- Support structured daily English learning (18-minute sessions)
- Preserve quote authenticity while making content accessible

## Main Sections

### 1. Inspirational Quotes
- Collection organized by themes/categories
- Proper attribution maintained
- Educational and motivational focus

### 2. English Learning
Subdirectories:
- `Foundations/` - Basic grammar (nouns, pronouns, sentence structure)
- `Grammar/` - Progressive lessons and exercises
- `Vocabulary/` - Alphabetically organized (A-Z)

### 3. Strategic Literacy
- Personal development content
- Life principles and discipline reflections

### 4. Scripts
- PowerShell verification tools
- Content management automation

## Content Standards

- Every quote should inspire or teach
- Vocabulary definitions must be practical and learnable
- Grammar examples from real quotes
- 18-minute daily learning focus

## File Organization

- Vocabulary: `EnglishLearning/Vocabulary/A.md`, `B.md`, etc.
- Grammar: `EnglishLearning/Grammar/*.md`
- Quotes: `InspirationalQuotes/*.md`
- Scripts: `Scripts/*.ps1`

## Core Principles

- Maintain inspirational and educational value
- Keep vocabulary definitions clear and practical
- Preserve quote authenticity and attribution
- Support structured daily English learning
- Ensure content is accessible for learners
