# Cline rules — Inspirational Quotes & English Learning

Personal Cline configuration for the `inspirational-quotes` learning workspace.

## Layout

| Path | Role |
|------|------|
| `AGENTS.md` | Cline entry point — precedence, scope, pointers |
| `core-context.md` | Compact always-on repo context |
| `rules/` | Standing guardrails |
| `skills/` | On-demand procedures |
| `agents/` | Delegated task playbooks |
| `workflows/` | Step-by-step Cline workflows |

## Core principles

- Maintain inspirational and educational value
- Keep vocabulary definitions clear and practical
- Preserve quote authenticity and attribution
- Support structured daily English learning
- Ensure content is accessible for learners
