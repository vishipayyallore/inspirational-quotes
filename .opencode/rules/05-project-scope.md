# Project Scope

## In scope

- Curating inspirational quotes with attribution.
- Building English vocabulary (A–Z files, definitions, usage examples).
- Grammar study notes and exercises (Foundations → Grammer → advanced).
- Strategic literacy and personal-development content.
- PowerShell scripts for vocabulary verification and content management.
- Chat abbreviation reference.

## Out of scope

- Code projects, applications, or software development.
- Data science, ML, or any Python/Jupyter content.
- Third-party courseware or copyrighted educational materials (beyond fair-use quotes).
- Build systems, CI/CD, package management, or testing frameworks.
- Anything in `.archive/` — preserved, never modified.
- Anything in `.clinerules/` — Cline-specific agent config.
