# Educational Content Rules

**Context**: Personal English-learning workspace (inspirational quotes, vocabulary, grammar).
**Priority**: HIGH

## Vocabulary entry format

Every word in `EnglishLearning/Vocabulary/*.md` must follow:

```
## WordName

**Meaning**: Clear, practical definition.
**Usage**: "Example sentence from a quote." - Brief context explanation.
```

- Words in each letter file must be in **strict alphabetical order** (case-insensitive).
- `WordsToOrganize.md` holds unplaced words; entries there follow the same format.
- `VocabularyDefinitions.md` is a summary index, not authoritative for order.

## Quote formatting

- Blockquote (`>`) for the quote text.
- Italic attribution on its own line: `_— Author Name_`
- Attribution must be preserved when known.
- Organize by theme/category when grouping multiple quotes.

## Grammar / Foundations format

- H2 (`##`) for main topics, H3 (`###`) for subtopics.
- Use examples from actual quotes in the collection when possible.

## Content principles

- **Attribution first**: Never drop a known author/source.
- **Practical over complex**: Default to beginner-friendly explanations.
- **Contextual learning**: pair vocab and grammar with real quote examples.
- **18-minute daily sessions**: content should be digestible in short bursts.

## What to avoid

- Fabricated or artificial examples not grounded in real quotes.
- Overly technical linguistic jargon where plain English suffices.
