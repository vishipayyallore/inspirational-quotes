# File Naming Conventions

## Vocabulary files
- Location: `EnglishLearning/Vocabulary/`
- Format: Single uppercase letter + `.md` (e.g., `A.md`, `B.md`).
- Special files: `WordsToOrganize.md`, `VocabularyDefinitions.md`.

## Grammar files
- Location: `EnglishLearning/Grammer/`
- Format: `Day<N>.md`, `_Template.md`.

## Foundations files
- Location: `EnglishLearning/Foundations/`
- Format: PascalCase (e.g., `Nouns.md`, `Pronouns.md`).

## Strategic Literacy files
- Location: `Strategic-Literacy/`
- Format: `NN-descriptive-name.md` (e.g., `01-today-is-mine.md`).

## Scripts
- Location: `Scripts/`
- Format: PascalCase for PowerShell (e.g., `Verify-Vocabulary.ps1`).

## Verification logs
- Location: `Logs/`
- Format: `verify-YYYYMMDD-HHmmss.txt`.
